一个页面，完成 Markdown 编辑、预览与多人同步。

左侧是深色控制台风格编辑区，右侧是清晰的排版预览区。选择单人模式可本地创作，选择合作模式则可在房间中与其他成员共享同一份 Markdown 文档。

本地渲染 marked.js + highlight.js 在浏览器侧完成解析和代码高亮。
